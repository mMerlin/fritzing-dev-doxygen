Design Rule Checks.

Report conflicts with the defined printed circuit board design rules. Currently that is only the minimum separation between non electrically connected copper elements on each layer (side) of a single PCB in a sketch, and minimum separation of copper from the edge of the board.

boardsingle PCB within the sketch

◆ AlsoNet

const QString DRC::AlsoNet = "alsoNet"
static

Design Rules Checking The basic process is to render all of the copper on one layer for a single net (connected traces and pads) to a black and white image, then render all other copper for the layer to a separate image. Any pixels that are in both images indicate a collision. Repeat for each layer (top and bottom). Repeat for each net.
