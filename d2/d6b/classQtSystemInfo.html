QString QtSystemInfo::getSystemInformation ( QString  format = QString())
Parameters
formatA format of the information to be returned. The following strings will be replaced: SYS_NAME - replaced with system name. SYS_VERSION - replaced with system version. KERN_NAME - replaced with kernel name. KERN_VERSION - replaced with kernel version. ARCH - replaced with architecture. If no format given, the method will default to: SYS_NAME SYS_VERSION (KERN_NAME KERN_VERSION ARCH).
